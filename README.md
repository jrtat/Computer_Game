# 人工智能结课作业代码开源

## 介绍

- HEX棋ai, 以一个当前棋局为输入, 以下一步棋的策略(采用(x,y)坐标的格式)为输出

- 主体部分采用带有UCB优化的MCTS, 并在“扩展”和“模拟”过程加入适当优化
   
> 开发过程详见：[github仓库](https://github.com/jrtat/Computer_Game) （基本只是当作网盘用, 开发时会定期上传当前进度的代码）
